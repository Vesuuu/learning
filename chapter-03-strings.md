## Теория

### Что это такое

**Строка (`str`)** — тип данных для текста: имена, пароли, сообщения, пути к файлам, ответы от API, содержимое HTML.

В Python строка — это **последовательность символов**. Каждый символ — буква, цифра, пробел, знак препинания, эмодзи и т.д.

```python
name = "Алексей"
message = 'Привет, мир!'
multiline = """Первая строка
Вторая строка"""
```

Три способа создать строку:
- `"двойные кавычки"`
- `'одинарные кавычки'`
- `"""тройные"""` — удобно для многострочного текста

Внутри строки с `"` можно использовать `'`, и наоборот:

```python
print("It's a sunny day")    # одинарная внутри двойных
print('Он сказал: "Привет"') # двойные внутри одинарных
```

**Экранирование** — спецсимволы через обратный слэш `\`:

| Последовательность | Значение |
|--------------------|----------|
| `\n` | Перенос строки |
| `\t` | Табуляция |
| `\\` | Сам символ `\` |
| `\"` | Кавычка `"` внутри `"..."` |
| `\'` | Кавычка `'` внутри `'...'` |

```python
print("Строка 1\nСтрока 2")
# Строка 1
# Строка 2
```

**Сырые строки (raw strings)** — `r"..."` — `\n` **не** превращается в перенос (полезно для путей Windows):

```python
path = r"C:\Users\name\file.txt"   # обратные слэши как есть
```

### Строка — последовательность: индексы и срезы

Строка ведёт себя как «коробка с пронумерованными ячейками». Нумерация **с нуля**:

```python
word = "Python"
#       P  y  t  h  o  n
#       0  1  2  3  4  5   ← положительные индексы
#      -6 -5 -4 -3 -2 -1   ← отрицательные (с конца)
```

| Индекс | `0` | `1` | `2` | `3` | `4` | `5` |
|--------|-----|-----|-----|-----|-----|-----|
| Символ | P | y | t | h | o | n |

```python
print(word[0])    # P — первый символ
print(word[-1])   # n — последний символ
print(word[2])    # t
```

**Срез (slice)** — взять **кусок** строки: `строка[начало:конец:шаг]`

| Синтаксис | Что делает |
|-----------|------------|
| `s[2:5]` | С индекса 2 до 4 включительно (5 **не** входит!) |
| `s[:3]` | С начала до индекса 2 |
| `s[3:]` | С индекса 3 до конца |
| `s[:]` | Копия всей строки |
| `s[::2]` | Каждый второй символ |
| `s[::-1]` | Строка **наоборот** |

```python
text = "Программирование"
print(text[0:5])    # Програ
print(text[5:])     # ммирование
print(text[-5:])    # ание — последние 5 символов
print(text[::-1])   # еинавориммаргорП
```

**Правило, которое путают чаще всего:** правая граница среза **не включается**. Как у `range()`: `range(2, 5)` → 2, 3, 4.

**Длина строки** — `len(s)`:

```python
len("hello")   # 5
len("")        # 0
len("Привет")  # 6 — каждая буква = один элемент (для кириллицы в Python 3)
```

### Строки неизменяемы (immutable)

Нельзя сделать `s[0] = "X"` — получишь `TypeError`. «Изменение» = **создание новой** строки:

```python
s = "hello"
s = "H" + s[1:]    # Hello — новая строка, старая "hello" осталась в памяти
# s[0] = "H"       # ОШИБКА!
```

Почему это важно:
- Методы вроде `.upper()` **не меняют** исходную строку — возвращают **новую**.
- Срезы тоже создают новую строку.

### Методы строк — главный инструмент

**Метод** — функция, «привязанная» к объекту. Вызывается через точку: `строка.метод()`.

Методы `str` **не изменяют** исходную строку (immutable!) — возвращают результат.

#### Регистр

| Метод | Что делает |
|-------|------------|
| `.upper()` | ВСЕ БУКВЫ ЗАГЛАВНЫЕ |
| `.lower()` | все буквы строчные |
| `.title()` | Каждое Слово С Заглавной |
| `.capitalize()` | Только первай символ заглавный |
| `.swapcase()` | Инвертирует регистр |

```python
name = "иван петров"
print(name.title())       # Иван Петров
print(name.upper())       # ИВАН ПЕТРОВ
print(name)               # иван петров — исходная не изменилась!
```

#### Пробелы и обрезка

| Метод | Что делает |
|-------|------------|
| `.strip()` | Убирает пробелы/переносы **с краёв** |
| `.lstrip()` | Только слева |
| `.rstrip()` | Только справа |

```python
raw = "   hello world   \n"
print(f"|{raw.strip()}|")    # |hello world|
```

`.strip("x")` — убирает указанные символы с краёв, не только пробелы:

```python
"...python...".strip(".")    # python
```

#### Поиск и проверка

| Метод | Что делает |
|-------|------------|
| `.find(подстрока)` | Индекс первого вхождения или `-1` |
| `.count(подстрока)` | Сколько раз встречается |
| `.startswith(префикс)` | Начинается ли с... → `bool` |
| `.endswith(суффикс)` | Заканчивается ли на... → `bool` |
| `.replace(старое, новое)` | Заменить подстроку |
| `.split(разделитель)` | Разбить в **список** |
| `"sep".join(список)` | Склеить список в строку |

```python
email = "user@mail.com"
print(email.find("@"))           # 4
print(email.endswith(".com"))    # True

text = "яблоко,груша,слива"
fruits = text.split(",")         # ['яблоко', 'груша', 'слива']
back = "-".join(fruits)          # яблоко-груша-слива
```

#### Проверки содержимого (валидация ввода)

| Метод | True, если... |
|-------|---------------|
| `.isdigit()` | Только цифры (`"42"` ✅, `"42a"` ❌) |
| `.isalpha()` | Только буквы |
| `.isalnum()` | Буквы и/или цифры |
| `.isspace()` | Только пробельные символы |
| `.islower()` / `.isupper()` | Весь регистр строчный/заглавный |

```python
"12345".isdigit()     # True
"123a".isdigit()      # False
"Привет".isalpha()    # True (кириллица — буквы)
```

**Важно:** `"False".isdigit()` → `False`, но `bool("False")` → `True` (непустая строка!). Разные вещи.

#### Оператор `in` — «содержится ли»

```python
"py" in "Python"      # True
"Java" in "Python"      # False
"@" in "user@mail.com"  # True
```

### Форматирование строк — как вставить переменные в текст

Три способа (от лучшего к устаревшему):

#### 1. f-strings (Python 3.6+) — **используй это**

```python
name = "Анна"
age = 25
print(f"Привет, {name}! Тебе {age} лет.")
# Привет, Анна! Тебе 25 лет.

price = 19.5
print(f"Цена: {price:.2f} руб.")   # Цена: 19.50 руб. — 2 знака после точки
```

Выражения внутри `{}`:

```python
a, b = 10, 3
print(f"{a} + {b} = {a + b}")   # 10 + 3 = 13
```

#### 2. `.format()`

```python
print("Привет, {}! Тебе {} лет.".format(name, age))
print("Привет, {n}! Тебе {a} лет.".format(n=name, a=age))
```

#### 3. `%` — старый стиль (знай, что существует)

```python
print("Привет, %s! Тебе %d лет." % (name, age))
```

**На Junior и в 2026 году — f-strings в 95% случаев.**

### Конкатенация: `+` vs `join`

**Склеивание через `+`:**

```python
greeting = "Привет, " + name + "!"
```

Работает, но в цикле на больших данных — медленно (каждый `+` создаёт новую строку).

**Правильный способ склеить много кусочков:**

```python
words = ["Python", "это", "круто"]
sentence = " ".join(words)    # Python это круто
```

**Правило:** много строк в одну → `join`. Две-три строки → `+` или f-string.

---

### Как это работает «под капотом»

#### Строка в памяти

В Python 3 строки хранят **Unicode** — символы практически любых языков и эмодзи: `"Привет 🐍"`.

```python
s = "Привет"
# В памяти: объект str, неизменяемый, знает свою длину
```

При `s1 + s2` Python создаёт **новый** объект `str`, копируя символы из обоих.

#### Индексация — O(1)

Доступ `s[i]` по индексу — мгновенный (константное время): Python знает, где начинается строка и смещение символа.

#### Срезы — создают копию

`s[2:5]` — новая строка из 3 символов. Исходная `s` не трогается.

#### Методы — вызывают C-код внутри CPython

`.upper()`, `.split()` и т.д. реализованы на C — быстрые. Но всё равно возвращают **новую** строку.

#### `join` быстрее `+` в цикле — почему

```python
# Медленно (O(n²) по памяти):
result = ""
for word in words:
    result += word + " "

# Быстро (O(n)):
result = " ".join(words)
```

Каждый `+=` создаёт новую строку и копирует всё накопленное. `join` один раз выделяет память под итог.

#### Итерация по строке

```python
for char in "Python":
    print(char)
```

Строка — **итерируемый** объект: на каждой итерации отдаёт один символ (строка длины 1).

| Сценарий | Что используешь |

### Микро-моменты, нюансы, частые ошибки новичков

#### 1. Строка vs число

```python
"5" + "3"      # "53" — склеивание, не 8!
int("5") + 3   # 8
```

#### 2. `input()` + сравнение с числом

```python
age = input("Возраст: ")   # str!
if age > 18:                 # TypeError!
if int(age) > 18:            # правильно
```

#### 3. Методы не меняют исходник

```python
s = "hello"
s.upper()
print(s)           # hello — не изменилось!
s = s.upper()      # правильно: HELLО
print(s)
```

#### 4. `find` vs `index`

- `.find(x)` → индекс или `-1`
- `.index(x)` → индекс или **ошибка** `ValueError`

Для новичка безопаснее `find` или `in`.

#### 5. Пустая строка — falsy

```python
name = ""
if name:
    print("есть имя")
else:
    print("имя пустое")   # сюда
```

#### 6. Срез за границей — не падает

```python
s = "hello"
print(s[100])      # IndexError — один индекс
print(s[2:100])    # llo — срез мягко обрезается
```

#### 7. `split()` без аргумента

```python
"a  b   c".split()     # ['a', 'b', 'c'] — любое кол-во пробелов
"a  b   c".split(" ") # ['a', '', 'b', '', '', 'c'] — пустые куски!
```

#### 8. `bool("False")` — ловушка!

```python
bool("False")    # True! — непустая строка
bool("")         # False
```

Строка `"False"` — это не логическое `False`.

#### 9. Сравнение строк — регистр важен

```python
"Python" == "python"   # False
"Python".lower() == "python".lower()  # True
```

#### 10. f-string: кавычки внутри

```python
print(f'Он сказал: "Привет"')           # одинарные снаружи
print(f"Имя: {name}")                   # двойные снаружи
```

#### 11. Иммутабельность и «изменение» в цикле

```python
# Нельзя собирать строку через присваивание по индексу
# Можно через += или join
```

#### 12. `len()` и пробелы

```python
len(" a ")    # 3 — пробелы считаются!
" a ".strip() # "a" → len 1
```

### Пример 8: replace и split/join

```python
# replace — замена подстроки
text = "Я люблю Java. Java — хороший язык."
new_text = text.replace("Java", "Python")
print(new_text)
print(text)   # исходник цел

# replace с лимитом — только первое вхождение
print(text.replace("Java", "Python", 1))

# split — строка → список
csv_line = "яблоко,банан,вишня"
fruits = csv_line.split(",")
print(fruits)           # ['яблоко', 'банан', 'вишня']

sentence = "Python is awesome"
words = sentence.split()    # без аргумента — по пробелам
```

```python
def is_palindrome(s):

    cleaned = s.lower().replace(" ", "")   # нижний регистр, убрали пробелы

tests = [

    "hello",
```

### Пример 16: Мини-программа — нормализация и проверка пароля

```python
def check_password(password):
    """Проверяет пароль, возвращает список ошибок."""
    errors = []

    if len(password) < 8:
        errors.append("Минимум 8 символов")

    if password.lower() == password:
        errors.append("Нужна хотя бы одна заглавная буква")

    if password.upper() == password:
        errors.append("Нужна хотя бы одна строчная буква")

    if not any(c.isdigit() for c in password):
        errors.append("Нужна хотя бы одна цифра")

    if " " in password:
        errors.append("Пробелы запрещены")

    return errors

# Тестируем
passwords = ["short", "alllowercase1", "ALLUPPERCASE1", "Good Pass 1", "GoodPass1"]

for pwd in passwords:
    errs = check_password(pwd)
    if errs:
        print(f"'{pwd}' → ОТКЛОНЁН: {', '.join(errs)}")
    else:
        print(f"'{pwd}' → ПРИНЯТ")
```

**Вывод консоли:**

```
'short' → ОТКЛОНЁН: Минимум 8 символов, Нужна хотя бы одна заглавная буква, Нужна хотя бы одна цифра
'alllowercase1' → ОТКЛОНЁН: Нужна хотя бы одна заглавная буква
'ALLUPPERCASE1' → ОТКЛОНЁН: Нужна хотя бы одна строчная буква
'Good Pass 1' → ОТКЛОНЁН: Пробелы запрещены
'GoodPass1' → ПРИНЯТ
```

**Разбор `any(...)` простыми словами:**  
`any(c.isdigit() for c in password)` — «есть ли **хотя бы один** символ `c`, для которого `c.isdigit()` True?».  
Похоже на цикл с флагом, только короче. Полный разбор — **глава 7** (генераторы). Пока можно заменить на цикл:

```python
has_digit = False
for c in password:
    if c.isdigit():
        has_digit = True
        break
```

### FAQ начинающего

**В: Чем `find` отличается от `index`?**  
`find` → `-1` если нет. `index` → ошибка. Для новичка: `if "@" in email` безопаснее.

**В: Как убрать все пробелы, не только с краёв?**  
`s.replace(" ", "")` — убирает **все** пробелы. `.strip()` — только с краёв.

**В: `"5" * 3` — что это?**  
`"555"` — умножение строки (повторение). `5 * 3` → `15` (числа).

**В: Как получить первый/последний символ безопасно?**  
`s[0]` и `s[-1]` если строка не пустая. Сначала `if s:` или `if len(s) > 0`.

**В: f-string или .format()?**  
f-string. Точка.

**В: Почему `split(" ")` даёт пустые строки?**  
Потому что два пробела подряд = «пустой кусок между разделителями». Без аргумента `split()` умнее.

**В: Как сравнить строки без учёта регистра?**  
`s1.lower() == s2.lower()` или `s1.casefold() == s2.casefold()` (casefold — агрессивнее для Unicode).

**В: Строка `"None"` и значение `None` — одно и то же?**  
Нет! `"None"` — строка из 4 символов. `None` — отдельный тип «ничего».
