# ЧАСТЬ 1. Установка

## 1.1. Скачать Python

1. Открой [https://www.python.org/downloads/](https://www.python.org/downloads/)
2. Скачай **Python 3.12** или новее (3.11+ подойдёт).
3. При установке **обязательно** поставь галочку:
   - **Add python.exe to PATH** (добавить в PATH)

Без PATH команда `python` в терминале не найдётся.

---

## 1.2. Проверка установки

Открой **PowerShell** или **cmd** и введи:

```bash
python --version
```

**Должно вывести:** `Python 3.12.x` (или твоя версия)

Если «команда не найдена» — чеклист:

1. Переустанови Python → галочка **Add python.exe to PATH** → Install.
2. Закрой **все** окна терминала и Cursor, открой заново.
3. Снова `python --version`. Не помогло → в установщике выбери **Repair**.
4. Крайний случай: полный путь  
   `C:\Users\Имя\AppData\Local\Programs\Python\Python312\python.exe --version`

# ЧАСТЬ 2. Первый запуск

## 2.1. REPL — интерактивный Python

В терминале:

```bash
python
```

Появится `>>>`. Введи:

```python
>>> print("Привет!")
>>> 2 + 2
>>> exit()
```

**Разбор:** REPL — «калькулятор с памятью». Удобно для экспериментов. Для домашки — файлы `.py`.

---

## 2.2. Первый файл

Создай `hello.py`:

```python
print("Я запустил Python!")
print(10 + 5)
```

В терминале перейди в папку с файлом:

```bash
cd C:\Users\Имя\python-learning
python hello.py
```

**Должно вывести:**
```
Я запустил Python!
15
```

## 2.4. Типичные ошибки новичка

| Ошибка | Причина | Что делать |
|--------|---------|------------|
| `SyntaxError` | Опечатка, забытая `:` или скобка | Читай строку из сообщения, проверь отступы |
| `IndentationError` | Пробелы и табы смешаны | В Python 3 — **4 пробела** на уровень |
| `NameError: name 'x' is not defined` | Переменная не создана | Напиши `x = ...` до использования |
| `python не найден` | Нет в PATH | Переустанови с галочкой PATH |
| Файл не найден | Не та папка в `cd` | `dir` (Windows) — посмотри файлы в папке |

**Сообщение об ошибке читай снизу вверх:** последняя строка — тип ошибки, выше — твой код.
